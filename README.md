== Description ==

CSSで作られたわぷーをPolymerで使えるようにコンポーネント化しました。

== 使い方 ==

Polymer1.0が使用できることが前提です。
```
<!-- コンポーネントをインポートする -->
<link rel="import" href="css-wapuu.html">
<!-- 表示したい場所にタグを記述する -->
<css-wapuu></css-wapuu>
```

== 注意 ==

Polymerはversion1.0、bowerを使って構築されていることを想定しています。
